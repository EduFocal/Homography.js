Homography.js

Homography.js class. It implements the Homography class, designed for implementing image homographies in Javascript. It is designed to be easy-to-use (even for developers that are not familiar with Computer Vision) lightweight and fast, in order to execute in real time applications (even in low-spec devices such as budget smartphones).
